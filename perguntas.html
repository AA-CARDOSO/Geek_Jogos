1. Qual é o objetivo do site?

O site da Geek Jogos tem como objetivo trazer noticias atualizadas de jogos eletrônicos, tal como lançamentos, atualizações e entre outros.

2. Posso comprar jogos pelo site?

Não, o site da Geek Jogos vai indicar uma plataforma para a realização da compra do jogo.

3. As gameplay são realizadas pela Geek Jogos?

Não, as gameplay são realizadas por parceiros da Geek Jogos.

4. Aonde é disponível as gameplay?

As gameplay são disponíveis na plataforma do YouTube.

5. Como faço para visualizar as gameplay?

Abaixo das informações de cada jogo, você irá encontrar um botão para lhe redirecionar a plataforma do YouTube, no qual você poderá visualizar ás gameplays.

6. Como faço para comprar os jogos?

Para comprar os jogos, você precisa clicar no botão que se encontra logo abaixo das informações do jogo, no qual irá lhe redirecionar para a plataforma de venda.
